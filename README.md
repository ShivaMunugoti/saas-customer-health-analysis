SaaS Customer Health & Churn Analysis Dashboard

Tools Used
Power BI
Excel

Project Overview
This project analyzes SaaS customer engagement and identifies churn risks using login activity, support tickets, and renewal timelines.

Key Metrics
• Total Monthly Revenue
• Customer Distribution by Plan
• Revenue by Plan
• Customer Engagement (Login Activity)

Key Insights
• Enterprise plan generates the highest revenue.
• Customers with low login activity show higher churn risk.
• Engagement metrics help identify at-risk customers before renewal.

Dashboard
Power BI dashboard visualizes customer behavior, revenue performance, and churn risk indicators.
